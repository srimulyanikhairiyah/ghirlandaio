# Alasan Korporasi Teknologi Modern Mengadopsi Model Open Source
## Pendahuluan
Berbicara tentang model open source, yaitu model pengembangan perangkat lunak yang kode sumbernya dapat diakses dan dikembangkan  terbuka. Walaupun terlihat bertentangan dengan tujuan bisnis yang mencari keuntungan, pada kenyataannya model ini justru banyak digunakan oleh korporasi.
